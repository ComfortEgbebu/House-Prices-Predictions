Machine Learning project that utilizes historical data to identify patterns and trends in house prices. 
The aim is to assist Maple Valley Realty to develop an accurate Predictive Model to estimate house prices based on property features and market data.

Accurate price predictions will help the company to:
Advise sellers on optimal pricing strategies
Provide buyers with fair market value estimates
Assist investors in identifying undervalued properties

LinearRegression Model was used to achieve this goal. The model has R2 of 86.7% indicating very high reliability and model accuracy.
